# Сервис генерации кода для встраиваемых систем на основе ИИ.

## Требования:
1. Запущена ollama с моделью llama3 порт 11434
2. Заполнен config.py
3. Установлен uv
4. Установлены зависимости

# Запуск:
1. uv run main.py

## Интерфейс и использование:
1. http://127.0.0.1:8000/ - основное меню
2. http://127.0.0.1:8000/emb_code_gen - апи генерации
