# Сервис генерации кода для встраиваемых систем на основе ИИ.

## Требования:
1. Запущена ollama с моделью llama3 порт 11434
2. Заполнен config.py
3. Установлен pip
4. Установлены зависимости

# Подготовка СУБД
1. python init_database.py

# Запуск:
1. pip install -r requirements.txt
2. python main.py

## Интерфейс и использование:
1. http://127.0.0.1:8000/ - страница Авторизация
2. http://127.0.0.1:8000/auth - api авторизации
3. http://127.0.0.1:8000/code_generator - основное меню генерации
4. http://127.0.0.1:8000/emb_code_gen - api генерации
5. http://127.0.0.1:8000/admin_console - консоль администратора
6. http://127.0.0.1:8000/create_user - api создания пользователя
7. http://127.0.0.1:8000/audit - старница Аудит
8. http://127.0.0.1:8000/dashboard - страница Обзор системы
9. http://127.0.0.1:8000/deploy - страница/api развертывания
10. http://127.0.0.1:8000/docs - swagger ui
11. http://127.0.0.1:8000/metrics - prometheus metrics

## Ролевая модель
1. user - Пользователь с правами генерации кода, развертывания и просмотра обзора системы
2. admin - Администратор, не имеет прав генерации, имеет права на редактирование справочника технологий и языков
3. viewer -  Наблюдатель, имеет специальный интерфейс для контроля состояния системы
4. auditor - Аудитор, имеет возможность просматривать Журнал Аудита
